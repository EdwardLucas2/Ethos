# Ethos: Evidence Upload Screen - Functional Specification ## User Stories * "As a participant, I want a fast way to capture and submit proof of my habit so it doesn't interrupt my routine." * "As a participant, I want to confirm my photo looks good before uploading it." * "As a participant, I want to add an optional note to my proof to provide context to the group." * "As a user, I want to know that my upload is being verified via GPS and timestamp for maximum accountability." ## Features * **Live Camera View:** A framed viewport for capturing real-time evidence. * **Retake Action:** A simple button to clear the current photo and try again. * **Note Input:** A text area for optional "deliverable" explanations. * **Automated Metadata:** Read-only displays for current Timestamp and Location status (e.g., "VERIFIED_GPS"). * **Primary Upload CTA:** A massive button to trigger the smart contract verification protocol and submit the proof.
