# Ethos: Results Incoming (Last Week) - Functional Specification ## User Stories * "As a participant, I want to see the final standings from the previous cycle while it's still being resolved." * "As a voter, I want to know exactly how many items I need to review to finalize the stakes." * "As a user, I want to see a summary of everyone's final verified vs pending progress." ## Features * **Period Switcher:** Top toggle set to "LAST WEEK" for retrospective view. * **Final Standings List:** A ranked list of participants showing their final counts (e.g., "3/3 VERIFIED") and success/fail badges. * **Verification CTA Card:** A prominent blue block showing "VERIFICATION PENDING" and the number of items remaining to be reviewed, with a "REVIEW NOW" link. * **Contract Specs Recap:** A summary of everyone's habits and the end date for that specific cycle. * **Upload Status Lock:** A footer notification clarifying that the submission window is closed.
