# Ethos: Contract Overview (Active Cycle) - Functional Specification ## User Stories * "As a participant, I want to see my progress vs my friends' progress in real-time." * "As a participant, I want to see exactly what I have 'On the Line' (the forfeit) to keep the stakes high." * "As a participant, I want to know how much time is left in the current cycle so I can plan my remaining actions." * "As a user, I want to quickly jump between the current week and last week's results." ## Features * **Period Switcher:** A top-level toggle for "THIS WEEK" vs "LAST WEEK". * **Stakes Card:** A high-contrast display of the shared forfeit (e.g., "A PINT"). * **Cycle Timeline:** A progress bar showing time elapsed from Start Date to End Date. * **Progress Arena:** A list of participants with: * **Habit & Frequency Label:** e.g., "GYM 3X/WEEK (2/3)". * **Multi-tone Progress Bar:** Distinguishing between verified (solid) and pending (patterned) progress. * **Proof Feed Preview:** A horizontal carousel of the latest photo uploads with a "View All" link. * **Primary Action (FAB/Banner):** A persistent "SNAP PROOF" button to trigger the upload flow.
